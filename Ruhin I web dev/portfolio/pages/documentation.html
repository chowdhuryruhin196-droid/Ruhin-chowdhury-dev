‚ùì Frequently Asked Questions

How do I add images?

Download images from Unsplash, Pexels, or Pixabay, then save them to the portfolio/images/ folder with the filenames mentioned in the Getting Started section.

Why aren't my images showing?

Check the following:

  • Image filename matches exactly (case-sensitive)
  • Image is in the images/ folder
  • Clear your browser cache (Ctrl+Shift+Del)
  • Refresh the page

How do I change the colors?

Edit the CSS variables in styles.css at the top of the file (around line 6). Change the color hex codes to your preferred colors.

Can I add more pages?

Yes! Create a new HTML file in the pages/ folder following the same structure as the existing pages. Don't forget to:

  • Include the same header and footer
  • Link the CSS and JavaScript files
  • Add navigation links to other pages

How do I make the form actually send emails?

Currently, the form validates but doesn't send emails. To enable email functionality, use:

  • Formspree: formspree.io (Free tier available)
  • Firebase: firebase.google.com (Free tier available)
  • Netlify Forms: Built-in if deployed on Netlify

Is this portfolio responsive?

Yes! The portfolio is fully responsive with 4 breakpoints:

  • Desktop: 1024px and above
  • Tablet: 768px to 1023px
  • Mobile: 480px to 767px
  • Small Mobile: Below 480px

What browsers does this work on?

This portfolio works on:

  • Chrome/Edge (latest)
  • Firefox (latest)
  • Safari (latest)
  • Mobile browsers (iOS Safari, Chrome Mobile)

Can I use my own domain?

Yes! Depending on your hosting:

  • GitHub Pages: Can connect custom domain in settings
  • Netlify: Easy custom domain setup
  • Traditional Hosting: Set up in domain registrar

How do I improve SEO?

The portfolio already includes basic SEO best practices. To enhance it:

  • Add descriptive alt text to images
  • Update meta descriptions in each page
  • Create a sitemap.xml file
  • Add social media meta tags (OpenGraph)
  • Keep content updated regularly

Is the portfolio free to use?

Yes! This portfolio is completely free to use, modify, and deploy. All third-party resources (fonts, images) are also free or open-source.

How do I contact support?

This is your portfolio! You can:

  • Check the documentation in this Resources page
  • Review the comments in HTML/CSS/JavaScript files
  • Check the README and other guide files
